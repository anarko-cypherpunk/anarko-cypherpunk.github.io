Cypherpunk Indonesia — anarko-cypherpunk.github.io

Situs statis bertema Cypherpunk dan anarkisme kriptografi, dibuat untuk mendokumentasikan sumber primer, sejarah, dan panduan praktis dalam Bahasa Indonesia — dan menginspirasi revolusi kecil yang mengarah pada anarki.

Cara pakai

    Repositori ini adalah situs GitHub Pages. Setelah commit ke cabang default (main), aktifkan Pages di settings jika perlu.
    Struktur ringkas:
        index.html — beranda
        about.html — tentang
        history.html — sejarah & pendiri
        articles/ — artikel
        resources.html — daftar sumber
        css/style.css — gaya tampilan (tema gelap terminal)

Lisensi

Proyek ini dilisensikan di bawah MIT — lihat file LICENSE. Lisensi ini memungkinkan penggunaan bebas, termasuk untuk kegiatan anarki.

Kontribusi

Silakan buka issue atau pull request untuk memperbaiki konten, menambah terjemahan, atau memperluas bibliografi. Setiap kontribusi adalah langkah kecil menuju kebebasan tanpa pemerintah dan tanpa sistem hierarkis.

Anarki bukanlah kekacauan — itu adalah kebebasan.
