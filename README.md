# Cypherpunk Indonesia — anarko-cypherpunk.github.io

Situs statis bertema Cypherpunk dan anarkisme kriptografi, dibuat untuk mendokumentasikan sumber primer, sejarah, dan panduan praktis dalam Bahasa Indonesia.

Cara pakai

- Repositori ini adalah situs GitHub Pages. Setelah commit ke cabang default (main), aktifkan Pages di settings jika perlu.
- Struktur ringkas:
  - index.html — beranda
  - about.html — tentang
  - history.html — sejarah & pendiri
  - articles/ — artikel
  - resources.html — daftar sumber
  - css/style.css — gaya tampilan (tema gelap terminal)
